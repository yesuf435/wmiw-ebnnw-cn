展示 CSS Grid 和 Flexbox 现代布局技术

1. Flexbox 弹性布局

Flexbox 适合一维布局，如导航栏、按钮组等。支持自动换行和灵活的空间分配。

2. CSS Grid 自适应布局

Grid 布局会自动根据容器宽度调整列数，每列最小宽度 250px。适合响应式卡片列表。

4. 响应式表单布局

表单使用 Grid 布局，桌面端双列，移动端自动切换为单列。

5. 响应式断点说明

桌面端（> 1024px）：

  • Grid 布局显示 4 列
  • 卡片显示 3 列
  • 表单显示 2 列

平板端（768px - 1024px）：

  • Grid 布局显示 3 列
  • 卡片显示 2 列
  • 表单显示 1 列

手机端（< 768px）：

  • Grid 布局显示 1 列
  • 卡片显示 1 列
  • 导航栏垂直堆叠
  • 间距自动缩小

6. 使用指南

在项目中使用这些布局：

  1. 在页面中引入 responsive-layout.css
  2. 选择合适的布局类名（如 grid-auto-fit, flex-container 等）
  3. 根据需要自定义 CSS 变量（颜色、间距等）
  4. 测试不同屏幕尺寸下的显示效果

推荐的开发工具：

  • Chrome DevTools 的设备模拟器
  • Firefox 的响应式设计模式
  • 真机测试（iOS Safari、Android Chrome）
